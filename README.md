# Team AI Assessment Tool

An interactive web-based survey tool to assess AI and automation literacy across a project team.

Built as part of an internal AI training programme at Chronicle to understand the team's current AI knowledge baseline and design targeted upskilling paths.

## What It Does

- Collects structured responses on team members' AI tool familiarity and usage patterns
- Covers areas: AI tools awareness, Python/automation experience, GIS workflow knowledge, and learning priorities
- Generates a team-level snapshot to guide training programme design
- Deployed as a static site via GitHub Pages — no backend required
